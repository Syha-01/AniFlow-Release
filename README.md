## Disclaimer:**
> **AniFlow does not host any anime content.** This application is strictly a user interface (client) that connects to third-party APIs. We do not store, distribute, or control any of the media files accessible through this app. All content is provided by external sources.
>
> If you wish to use this application, you must provide your own backend API or host your own instance of the [hianime-API](https://github.com/yahyaMomin/hianime-API.git).

AniFlow is a premium, feature-rich anime streaming application built with **Expo** and **React Native**. It provides a seamless experience for discovering, tracking, and watching your favorite anime titles with a modern and fluid user interface.

## API Setup (Required)

To use AniFlow, you need a running instance of the backend API.

1.  **Host the API**: You can host your own instance of the API by following the instructions at:
    [**https://github.com/yahyaMomin/hianime-API.git**](https://github.com/yahyaMomin/hianime-API.git)

2.  **Configure the App**:
    *   On the first launch (or in Settings), you will be prompted to enter your **API Base URL**.
    *   Enter the URL where you have deployed the hianime-API (e.g., `https://your-api-instance.vercel.app`).

## Features

- **Dynamic Discovery**: Spotlight carousel for featured anime and top-ten lists.
- **Advanced Search**: Real-time search suggestions and comprehensive filtering options.
- **Fluid Player**: High-quality video streaming with support for multiple servers and sub/dub options.
- **Release Schedule**: Stay updated with the latest episode releases through the integrated schedule.
- **Watchlist & History**: Save your favorite anime and keep track of your progress.
- **Deep Details**: Explore characters, voice actors, and related anime information.
- **Premium UI**: Glassmorphism effects, smooth animations, and a dark-themed aesthetic.
- **Dynamic Configuration**: Easily configure your own API base URL directly within the app.
